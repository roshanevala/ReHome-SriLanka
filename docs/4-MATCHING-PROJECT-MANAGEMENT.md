# Matching & Project Management

## Overview
This document describes how donors and beneficiaries are matched, and how construction projects are managed from funding to completion.

---

---

### Step 2: Donor Selection Options

#### Option A: Sponsor a Full House
- Donor commits to full estimated cost
- Becomes primary sponsor for that family
- Gets exclusive updates on construction
- Name displayed to family (if donor agrees)

**Process**:
1. Donor clicks "Sponsor This Family Fully"
2. System shows commitment form
3. Donor confirms budget allocation
4. Admin receives notification
5. Admin confirms match

#### Option B: Partial Sponsorship
- Donor contributes partial amount (e.g., LKR 300,000)
- Goes into funding pool for that specific family
- Multiple donors can fund one house
- Shared updates to all contributors

**Process**:
1. Donor clicks "Contribute LKR X to this family"
2. Enter contribution amount
3. System adds to family's funding total
4. When fully funded â†’ construction starts

#### Option C: Shared Pool (High Priority)
- Donor contributes to general fund
- Admin allocates to highest priority cases
- Donor receives summary reports of all funded houses
- Fastest way to help multiple families

**Process**:
1. Donor selects "Fund High Priority Pool"
2. Enter total contribution
3. Admin allocates based on priority scores
4. Donor gets monthly reports

---

### Step 3: Admin Confirmation

**Admin Reviews**:
- Donor's budget availability
- Family's current funding status
- Match appropriateness
- Donor preferences vs family profile

**Admin Actions**:
1. Log into matching dashboard
2. Review match request
3. Confirm or suggest alternative
4. Update both parties
5. Change status to "Match Confirmed"

**Automatic Updates**:
- **Beneficiary Status**: `Approved` â†’ `Matched - Pending Funds`
- **Donor Status**: `Active` â†’ `Funds Allocated`
- **Project Status**: Create new project record

---

### Step 4: Funding Confirmation

**Payment Process**:
1. Admin sends payment details to donor
2. Donor transfers funds (bank transfer preferred)
3. Admin confirms receipt
4. Issues official receipt with tax info (if applicable)
5. Records contribution in system

**Fund Holding**:
- Funds held in escrow/project account
- Released in phases (foundation, structure, finishing)
- Unused funds returned or reallocated (with donor approval)

**Status Update**:
- **Project Status**: `Matched - Pending Funds` â†’ `Funded - Planning`

## Project Management Workflow

### Phase 1: Planning (Week 1-2)

**Activities**:
1. **Site Assessment**
   - Measure land
   - Check soil condition
   - Verify utilities (water, electricity access)
   - Assess access for materials

2. **Design Selection**
   - Choose from standard designs (cost-effective)
   - Basic floor plan (2-3 bedrooms, kitchen, bathroom)
   - Adapt for family size
   - Get beneficiary approval

3. **Contractor Selection**
   - Get 2-3 quotes
   - Choose licensed contractor
   - Sign contract
   - Set timeline

4. **Permits & Approvals**
   - Building permit from local authority
   - Environmental clearance (if needed)
   - GN office approval

**Deliverables**:
- Approved design
- Signed contract
- Permits obtained
- Construction schedule

**Status**: `Funded - Planning` â†’ `Ready to Start`

---

### Phase 2: Construction (Week 3-16)

#### Milestone 1: Foundation (Week 3-5)
**Activities**:
- Clear site
- Mark foundations
- Dig foundation trenches
- Pour concrete foundation
- Cure concrete

**Photos**: Take 5-8 photos showing:
- Cleared site
- Foundation marking
- Concrete pouring
- Completed foundation

**Verification**: Admin or field officer visits to confirm

**Payment**: Release 25% of funds to contractor

**Status Update**: Milestone marked "Complete"

---

#### Milestone 2: Walls & Structure (Week 6-10)
**Activities**:
- Lay bricks/blocks
- Build walls to roof level
- Install pillar reinforcement
- Complete masonry work

**Photos**: Take 5-8 photos showing:
- Wall construction progress
- Structural elements
- Height progress
- Completed walls

**Verification**: Structural integrity check

**Payment**: Release 30% of funds

**Status Update**: Milestone marked "Complete"

---

#### Milestone 3: Roofing (Week 11-13)
**Activities**:
- Install roof structure
- Lay roofing sheets/tiles
- Seal and waterproof
- Install gutters

**Photos**: Take 5-8 photos showing:
- Roof frame
- Roofing material installation
- Completed roof
- Interior view

**Verification**: Check for leaks, proper installation

**Payment**: Release 20% of funds

**Status Update**: Milestone marked "Complete"

---

#### Milestone 4: Doors & Windows (Week 13-14)
**Activities**:
- Install door frames
- Hang doors
- Install window frames
- Fit windows with grills

**Photos**: Take 3-5 photos showing:
- Installed doors
- Fitted windows
- Security features

**Verification**: Check operation, security

**Payment**: Release 10% of funds

**Status Update**: Milestone marked "Complete"

---

#### Milestone 5: Finishing (Week 14-16)
**Activities**:
- Plastering walls
- Floor finishing
- Paint interior/exterior
- Electrical wiring
- Plumbing fixtures
- Final cleanup

**Photos**: Take 10-12 photos showing:
- Each room
- Kitchen
- Bathroom
- Exterior views
- Overall finish

**Verification**: Final inspection checklist

**Payment**: Release final 15% of funds

**Status Update**: Milestone marked "Complete"

---

## Handling Special Situations

### Project Delays
**Common Causes**:
- Weather (monsoon, flooding)
- Material shortages
- Contractor issues
- Permit delays
- Beneficiary requests

**Process**:
1. Contractor reports delay
2. Admin assesses reason
3. Update timeline
4. Notify donor
5. Document reason
6. Monitor closely

**Notification to Donor**:
"Project update: Construction delayed by 2 weeks due to heavy monsoon rains. New expected completion: Dec 15. We'll keep you updated."

---

### Cost Overruns
**If costs exceed estimate**:
1. Contractor provides revised estimate
2. Admin reviews necessity
3. Options:
   - Use contingency fund (if available)
   - Request additional funding from donor
   - Reduce non-essential features
   - Use pooled funds

**Notification to Donor**:
"We've encountered [reason]. Estimated additional cost: LKR 50,000. Would you like to cover this, or shall we adjust the plan?"

---

### Contractor Issues
**Red flags**:
- Missed milestones repeatedly
- Poor quality work
- Fund mismanagement
- Communication problems

**Actions**:
1. Document issues
2. Issue warning
3. If no improvement: terminate contract
4. Find replacement contractor
5. Update timeline
6. Inform donor

---

### Family Changes
**Possible scenarios**:
- Family moves away (rare)
- Family no longer needs house
- Additional family members
- Family disputes

**Process**:
1. Verify situation
2. Assess options:
   - Continue if temporary issue
   - Reassign to another family if permanent
   - Adjust design if needed
3. Get donor approval for changes
4. Document decision

---

## Quality Control

### Construction Standards
- Follow Sri Lankan building codes
- Use quality materials (approved suppliers)
- Licensed contractor required
- Regular inspections at each milestone
- Independent quality checks

### Photo Documentation Requirements
- Minimum 3 photos per milestone
- Show clear progress
- Include date stamps
- Multiple angles
- Before/after comparisons

### Inspection Checklist
Download at each milestone:
- [ ] Work matches approved design
- [ ] Materials quality verified
- [ ] Workmanship standard met
- [ ] Safety measures in place
- [ ] No shortcuts taken
- [ ] Timeline on track
- [ ] Budget on track

---

## Success Metrics

**Project Level**:
- On-time completion rate: Target 85%
- On-budget rate: Target 90%
- Quality satisfaction: Target 95% (family feedback)
- Donor satisfaction: Target 95%

**System Level**:
- Average time from match to handover: Target <16 weeks
- Average cost per house: Track and optimize
- Projects completed per month: Growth target
- Re-work rate: Target <5%
